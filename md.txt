проект newsSearch
разработка проекта на основе сборщика WebPack
Использованные технологии:
HTML;
CSS => Materialize;
JavaScript =>Синтаксис ES6,
 EventListener, #fragment, Ajax(GET / POST), API(newsapi.org)
Materialize -> preloader;
Функционал пользователя:
запросить у сервера по нажатию на кнопку
 существующие новости по выбранной стране(при первой загрузке Украина)
Осуществить поиск новстей по введенному значению, при отсутствие новостей
 по значению уведомлние о отсутсвии таких новостей
